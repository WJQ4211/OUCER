# 海大人校友论坛 - UI设计规范

## 一、设计系统概览

### 1. 品牌色彩体系
```
主色（海洋蓝）：#0066CC
辅助蓝：#E8F3FF
强调色（活力橙）：#FF6B35
成功绿：#2ECC71
警告黄：#F39C12
中性灰：#F5F5F5 / #666666 / #333333

渐变背景（头部）：#0066CC → #0052A3
```

### 2. 字体系统
- **标题（粗重）**：微软雅黑 Bold, 18px / 16px
- **正文**：微软雅黑 Regular, 14px
- **辅助文本**：微软雅黑 Regular, 12px
- **标签**：微软雅黑 Medium, 12px

### 3. 间距系统（基础单位 8px）
- XS: 4px | S: 8px | M: 12px | L: 16px | XL: 24px | XXL: 32px

## 四、常用组件库

### 1. 按钮设计

| 类型 | 样式 | 用途 |
|------|------|------|
| **主要按钮** | 蓝色实心 + 白字 | 确认、发布、提交 |
| **次要按钮** | 浅蓝背景 + 蓝字 | 取消、暂存 |
| **危险按钮** | 红色实心 + 白字 | 删除、下线 |
| **禁用按钮** | 灰色 + 灰字 | 不可操作 |

```
┌──────────────┐  ┌──────────────┐
│   确认发布    │  │   暂存       │
│ bg:#0066CC   │  │ border:#0066CC│
│ color:white  │  │ color:#0066CC│
└──────────────┘  └──────────────┘
```

## 五、交互动效

### 页面过渡
- 从下往上弹出：200ms
- 淡入/淡出：150ms cubic-bezier(0.4, 0, 0.2, 1)

### 反馈动效
- 按钮按下：按钮缩小 98% （50ms）
- 成功提示：绿色气泡弹出 + 3s 自动消失
- 加载中：骨架屏 + 脉冲动画

### 列表动画
- 进入列表：卡片从下往上依次出现（100ms 梯度延迟）
